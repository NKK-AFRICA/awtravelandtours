## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```

## Technologies Used

- **Vue.js 3** - Progressive JavaScript framework
- **Vue Router** - Official router for Vue.js
- **UIkit** - Lightweight and modular front-end framework
- **Vite** - Next-generation frontend build tool

## Customization

You can customize the website by:
- Updating images in the components
- Modifying content in the view files
- Adjusting styles using UIkit classes or custom CSS
- Adding new pages by creating new views and updating the router
